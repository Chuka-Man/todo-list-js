# Todo List App

Простой список задач на JavaScript.

Этот проект создан для практики работы с DOM, событиями и базовой логикой приложений.

## Возможности

* Добавление новых задач
* Отметка выполненных задач
* Удаление задач
* Работа с пользовательским вводом
* Обновление интерфейса без перезагрузки страницы

## Используемые технологии

* HTML
* CSS
* JavaScript

## Что я изучил

В процессе разработки проекта я отработал:

* DOM API
* addEventListener()
* Работа с input
* Создание и удаление элементов
* Обработка пользовательских действий
* Структура небольшого frontend-проекта

## Цель проекта

Разобраться, как JavaScript взаимодействует с интерфейсом пользователя и управляет элементами страницы.
